# Strapi Strarter with other args

version: 0.1.1

## Usage

In Strapi project folder  
```
starter.py [-h] [-p PORT] [strapi options]

optional arguments:
  -h, --help            show this help message and exit
  -p PORT, --port PORT  strapi server port
```
