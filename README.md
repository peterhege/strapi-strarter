# Strapi Strarter with other args
For https://strapi.io/  
version: 0.1

## Usage

Only for develop  
In Strapi project folder:  
```
starter.py [-h] [-p PORT] [strapi develop options]

optional arguments:
  -h, --help            show this help message and exit
  -p PORT, --port PORT  strapi server port
```
